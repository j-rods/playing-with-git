#Git Cheatsheet

###Staging area


Add files to staging area:

` git add `

Add all files to staging area:

` git add -A `



###Basic Branching

Create a new branch:

` git branch branchname `

Switch to branch:

` git checkout branchname `



Merging branch with master branch

` git checkout master `

` git pull origin master `

` git branch --merged `

` git merge branchname `

` git push origin master `
